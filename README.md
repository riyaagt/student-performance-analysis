# Student Performance Analysis

This project analyzes student performance data to explore how study habits, attendance, school type, and internet access relate to academic outcomes.

## Dataset
The dataset contains information about students including study hours, attendance percentage, school type, internet access, and subject scores.

## Key Insights
- Students who study more hours generally have higher attendance percentages.
- Attendance varies between public and private schools.
- Internet access shows an impact on students' study hours.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
